# FastVideo Consumer Guide

A standalone Next.js consumer guide for FastVideo configuration selection and advanced tuning.

## Requirements

- Node.js 20 or newer
- pnpm

## Run Locally

```bash
pnpm install
pnpm dev
```

Then open http://localhost:3000.

## Build

```bash
pnpm build
pnpm start
```

## Notes

This app is independent from the main FastVideo Python package. 
Run commands from this `consumer-guide/` directory.
